In this project an online store is presented.

Used software:
Angular
TypeScript
Java
mySQL

Backend:
Consists of and edge service and 4 proxy servers:
Users
Product
Cart
Transaction
There are two types of users, users and admins.

User:
There are two types of users, users and admins. Users would be the regular customers. They can create an account, save products in the cart, delte them and purchase them.
Admins can add products to the web and check the transactions that get created when a customer purchases a product.

Product:
Shows all the products by category (electronics, jewelery, men's clothes and women's clothes).

Cart:
Stores the client id and the products that are in their list when they log out.

Transaction:
This microservice saves the time of the purchase of the products by each client.
